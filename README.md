# 🎬 Movie Recommendation System

## 📌 Overview
This project is a **Movie Recommendation System** built using **Machine Learning** and deployed using **Streamlit**.

The system recommends movies similar to the one selected by the user using a **content-based filtering approach** powered by cosine similarity.

---

## 🚀 Features
- Interactive web interface using Streamlit  
- Recommends Top 5 similar movies  
- Fast response using precomputed similarity matrix  
- Clean and simple UI  
- Jupyter Notebook included for model building  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Streamlit
